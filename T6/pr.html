Abstract

Este projeto consistiu na criação de um site para gerir tarefas. As tarefas contêm um nome, um estado e uma data de término. O estado pode ser pendente ou concluído. A página inicial do site contém as tarefas agrupadas pelo seu estado e ordenadas pelo seu nome.

As operações possíveis de fazer com as tarefas são a sua criação, edição e remoção.

Em termos de implementação, o projeto consiste num servidor nodejs e json-server capazes de comunicar através de pedidos Http implementados pelo package axios. Incluí ainda formulários html que servem para obter dados do utilizador. Para suportar as operações possíveis, é criado um conjunto de "entry points" na Api que tratam os pedidos HTML de todos os tipos e geram o HTML necessário.

Deliverables

  • Ficheiro JSON com os dados das tarefas
