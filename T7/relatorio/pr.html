Abstract

Este projeto consistiu na criação de um site para gerir alunos. Os alunos contêm um identificador, o seu número, nome, o link para o repositório dos seus trabalhos de casa e os trabalhos de casa completados.

As operações possíveis de fazer com os tarefas são a sua criação, edição e remoção, incluindo os trabalhos de casa realizados.

Em termos de implementação, o projeto consiste num servidor nodejs e uma base de dados MongoDB capazes de comunicar através de o módulo mongoose. Incluí ainda páginas e formulários html gerados através do template pug que servem para mostrar e obter dados do utilizador. Para suportar as operações possíveis, é criado um conjunto de "entry points" na Api que tratam os pedidos HTML de todos os tipos e geram o HTML necessário.
