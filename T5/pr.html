Abstract

Este projeto consistiu na implementação de uma API de dados que permitisse retirar/alterar informação de uma base de dados de uma escola de música através do protocolo HTTP. Para isto, foi implementado um servidor em nodejs que recebe os pedidos HTTP do utilizador e os transforma em pedidos, também HTTP, à base de dados. Uma vez que o servidor obtém os dados pretendidos, gera com eles páginas HTML dinamicamente e envia-as para o utilizador. Os pedidos suportados pelo servidor são a listagem das páginas de alunos, cursos e instrumentos. Cada página de aluno, contém um link para o respetivo curso e em cada página de um curso, um link para o respetivo instrumento abordado no curso. Cada uma das páginas contém ainda um link que permite ao utilizador voltar à página inicial.

Os dados utilizados encontram-se num ficheiro Json pelo que é necessário utilizar um serviço como o json-server para que estes dados possam ser acedidos pelo servidor nodejs.
